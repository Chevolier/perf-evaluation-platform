EMD Web API 测试页面

API 配置

GET/ping - 健康检查

检查 API 服务是否正常运行

POST/emd_deploy - 部署模型

使用指定配置部署模型

POST/emd_status - 获取模型状态

检索模型的当前状态

POST/emd_destroy - 销毁模型

销毁指定的模型部署

POST/emd_supported_models - 获取支持的模型列表

获取 EMD 支持的所有模型列表（基础信息）

POST/emd_details - 获取模型详细信息

获取指定模型的详细信息

POST/emd_models - 列出已部署的模型

获取所有已部署模型的状态信息，包括正在进行中和已完成的部署
